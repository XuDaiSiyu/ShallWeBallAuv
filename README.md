# ShallWeBall
绍伟球-开源可编程水下机器人
## 架构
绍伟球分为硬件和软件两部分。  

软件：

1、驱动层（c，c++）  
2、底层固件（c，c++,asm）  
3、中间件（c）  
4、上层OS（funxr）（oc，swift，c，py）  
5、SDK（nodejs，ts）  

硬件： 

1、poix4  
2、nodefmu  
3、cahoti
